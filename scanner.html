// nmap + shodan recon tool v2.0
OFFENSIVE SECURITY PYTHON 3.8+ CLI LINUX / MACOS MIT
// DESCRIPTION
ThreatHasher is an offensive reconnaissance command-line tool designed to
automate the recon phase in security audits. It integrates Nmap and the Shodan API into
a single interactive workflow to discover ports, identify services, detect CVEs, and
export results in multiple formats for inclusion in reports.
// MODULES
[01]
NMAP SCAN
6 profiles. OS detection, versioning and NSE vuln scripts.
[02]
SHODAN
IP lookup, query search with filters, CVEs with CVSS.
[03]
DNS & PING
Forward/reverse DNS, subnet ping sweep, nslookup.
[04]
EXPORT
Complete JSON, CSV of CVEs, TXT report for documentation.
